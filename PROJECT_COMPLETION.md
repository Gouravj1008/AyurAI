## 🚀 Getting Started

### For Users (Fastest Way)

**Windows:**
```bash
1. Double-click: START_JARVIS.bat
2. Wait for setup (5-10 minutes first time)
3. Launch app automatically
```

**Mac/Linux:**
```bash
1. chmod +x start_jarvis.sh
2. ./start_jarvis.sh
3. App launches automatically
```

### For Developers

```bash
# Manual setup
python -m venv venv
venv\Scripts\activate  # or source venv/bin/activate
pip install -r requirements_complete.txt

# Configure
cp .env.example .env
# Edit .env and add Porcupine key

# Validate
python validate_system.py

# Launch
python run_jarvis_improved.py

# Or train custom model
python train_ayurveda_lora.py
```

## 🆘 Troubleshooting Quick Reference

| Issue | Quick Fix |
|-------|-----------|
| "PORCUPINE_ACCESS_KEY not set" | Get key from picovoice.ai, add to .env |
| "Vosk model not found" | Download from github.com/alphacep/vosk-models |
| Slow responses | First response is slow (normal), GPU option in .env |
| No microphone | Check system settings, use `AUDIO_DEVICE_INDEX=-1` in .env |
| CUDA errors | Set `USE_GPU=false` in .env |
| Module not found | Run: `pip install -r requirements_complete.txt` |
| App crashes | Check `jarvis.log`, run `python validate_system.py` |
